# Portfolio — Jolan Chappaz

Tu n'as besoin de modifier **qu'un seul fichier** : `projects.js`

## 📹 Vidéo d'accueil
Section `HOME` → change `src` avec le chemin de ta vidéo (ou image)

## ✏️ Page "About"
Section `ABOUT` → ta photo, ton texte, ton email et insta

## 🎨 Catégories / Projets
Section `CATEGORIES` → ajoute, supprime ou réorganise tes catégories.
Chaque slide : `{ type: "image", src: "..." }` ou `{ type: "text", content: "..." }`

## 🚀 Publier
```
git add .
git commit -m "Mise à jour"
git push
```
